À propos

Cette collection regroupe des mini outils développés rapidement pour répondre à des besoins temporaires et ponctuels. Ils partagent une nomenclature commune, une cohérence de code/stack, CSS et nommage pour être maintenus en commun.

Les outils sont soit des méthodes d'utilisation d'IA pour des mini projets, soit des mini projets générés via IA.

Caractéristiques communes

  • Design system unifié
  • Nomenclature cohérente
  • Stack technique standardisée
  • Déploiement via Coolify sur VPS

Collection open source - Vos données restent sur votre appareil
